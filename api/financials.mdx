# Financial statements API

Retrieve structured financial statement data for companies, including income statements, balance sheets, and cash flow statements.

<Note>
Requires authentication. Financial data is sourced from the `financial_data` schema in PostgreSQL.
</Note>

## Get income statement

Retrieve income statement data for multiple years.

**Endpoint**: `GET /companies/{symbol}/income-statement`

### Path parameters

<ParamField path="symbol" type="string" required>
  Company ticker symbol (e.g., "AAPL", "MSFT")
</ParamField>

### Query parameters

<ParamField query="years" type="integer" default={5}>
  Number of years of data to retrieve (1-10)
</ParamField>

### Response

<ResponseField name="symbol" type="string">
  Company ticker
</ResponseField>

<ResponseField name="data" type="array">
  Array of annual income statement records
  <Expandable title="Income statement fields">
    <ResponseField name="date" type="string">
      Statement date
    </ResponseField>
    <ResponseField name="calendarYear" type="integer">
      Calendar year
    </ResponseField>
    <ResponseField name="period" type="string">
      Period type (typically "FY" for fiscal year)
    </ResponseField>
    <ResponseField name="revenue" type="number">
      Total revenue
    </ResponseField>
    <ResponseField name="costOfRevenue" type="number">
      Cost of goods sold (COGS)
    </ResponseField>
    <ResponseField name="grossProfit" type="number">
      Gross profit
    </ResponseField>
    <ResponseField name="grossProfitRatio" type="number">
      Gross profit margin
    </ResponseField>
    <ResponseField name="researchAndDevelopmentExpenses" type="number">
      R&D expenses
    </ResponseField>
    <ResponseField name="operatingExpenses" type="number">
      Total operating expenses
    </ResponseField>
    <ResponseField name="operatingIncome" type="number">
      Operating income
    </ResponseField>
    <ResponseField name="operatingIncomeRatio" type="number">
      Operating margin
    </ResponseField>
    <ResponseField name="ebitda" type="number">
      Earnings before interest, taxes, depreciation, and amortization
    </ResponseField>
    <ResponseField name="ebitdaRatio" type="number">
      EBITDA margin
    </ResponseField>
    <ResponseField name="netIncome" type="number">
      Net income
    </ResponseField>
    <ResponseField name="netIncomeRatio" type="number">
      Net profit margin
    </ResponseField>
    <ResponseField name="eps" type="number">
      Earnings per share (basic)
    </ResponseField>
    <ResponseField name="epsDiluted" type="number">
      Earnings per share (diluted)
    </ResponseField>
  </Expandable>
</ResponseField>

### Example

```bash
curl https://api.financeagent.com/companies/AAPL/income-statement?years=5 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## Data source

Financial statement data is stored in PostgreSQL under the `financial_data` schema:

- `financial_data.income_statements` - Income statement records
- `financial_data.balance_sheets` - Balance sheet records
- `financial_data.cash_flow_statements` - Cash flow records

Data is filtered to show only annual (FY) periods by default, with the most recent statement per calendar year.
