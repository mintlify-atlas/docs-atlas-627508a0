## Overview

Real-Time News brings current market developments into Finance Agent using **Tavily search**. When you need information about very recent events (last few days/weeks) that haven't appeared in earnings calls or SEC filings yet, this feature searches live web sources and provides AI-generated summaries with article citations.

## How It Works

Real-Time News is automatically triggered when:
1. Your query contains news keywords ("latest news", "recent", "breaking")
2. The agent determines during iteration that news context would improve the answer

When activated:

1. **Query Tavily API** - Sends your question to Tavily's web search engine
2. **Retrieve Articles** - Gets top 5 most relevant news articles with:
   - Title
   - URL
   - Content excerpt
   - Published date
3. **Generate Summary** - Tavily's AI creates a concise summary of findings
4. **Format with Citations** - Articles are numbered [N1], [N2], [N3] for easy reference
5. **Merge into Answer** - News context is incorporated into the final response

## Features

### Automatic Semantic Routing

Finance Agent's question analyzer detects when news data would be valuable:

**Explicit News Queries** (always triggers news):
- "latest news"
- "recent developments"
- "breaking news"
- "what's happening with"

**Implicit News Needs** (triggered during iteration):
- Agent determines answer is incomplete without recent context
- Question asks about very recent events ("this week", "today")
- No relevant transcript/10-K data found

### Hybrid Mode

Finance Agent can combine news with other data sources:

**News + Transcripts:**
- Start with earnings call data for fundamentals
- Add news context for recent developments post-earnings

**News + 10-K:**
- Use 10-K for historical financial position
- Add news for recent strategic changes

**News Only:**
- Pure news queries focus exclusively on Tavily results

### Citation Format

News articles use **[N1], [N2], [N3]** markers:
- **[N1]** = First news article retrieved
- **[N2]** = Second news article
- Citations include title, source, published date, and URL

### AI-Generated Summaries

Tavily provides two types of output:
- **Answer**: AI-generated summary of all findings (concise overview)
- **Results**: Individual articles with titles, URLs, excerpts, and dates

Finance Agent incorporates both into the final response.

## Coverage

- **Sources**: Web-wide news search (financial news sites, press releases, company announcements, market analysis)
- **Freshness**: Real-time (typically minutes to hours old)
- **Results**: Top 5 most relevant articles per query
- **Provider**: Tavily API

## Technical Details

### Tavily Integration
```python
class TavilyService:
    def search_news(query: str, max_results: int = 5):
        # Query Tavily API
        response = tavily_client.search(
            query=query,
            max_results=max_results,
            include_answer="advanced"
        )
        
        return {
            "answer": "AI-generated summary",
            "results": [
                {
                    "title": "Article headline",
                    "url": "https://...",
                    "content": "Article excerpt",
                    "published_date": "2024-12-18",
                    "score": 0.95
                }
            ]
        }
```

### News Context Format

News is formatted with the same SOURCE pattern as transcripts/10-K:

```
=== LATEST NEWS ===

Summary: [Tavily AI-generated summary]

SOURCE [N1] Article Title (Published: Dec 18 2024, Source: https://...)
Article content excerpt (up to 800 characters)...

SOURCE [N2] Article Title (Published: Dec 17 2024, Source: https://...)
Article content excerpt...

=== END NEWS ===
```

This consistent format ensures the LLM cites news the same way as other sources.

    - "News from this week" (more targeted)

  </Step>

## Limitations

- **No Historical Archive**: Tavily focuses on recent/current news, not historical events
- **Web Content Variability**: Quality depends on available web sources at query time
- **No Structured Data**: News is unstructured text, not official financial filings
- **Source Reliability**: Tavily aggregates from multiple sources; verify important claims
- **No Real-Time Prices**: For stock prices, use a dedicated market data service

<Warning>
**Important**: News search does NOT provide real-time stock prices or market data. It searches for news articles and press releases only.
</Warning>

## Configuration

Real-Time News requires a Tavily API key:

```bash
TAVILY_API_KEY=your_tavily_api_key_here
```

If the key is not configured:
- News search will be disabled
- Queries will fall back to transcripts/10-K only
- A warning will appear in logs
